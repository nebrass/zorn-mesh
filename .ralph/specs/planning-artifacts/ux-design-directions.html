Local web UI · graphite + electric blue · React + Tailwind + Radix-style primitives

Six directions for making agent conversations visible.

These mockups explore different ways to express the zorn-mesh core interaction: open the local UI, see connected agents, click a trace, understand the whole agent conversation, and safely act when needed.

01 · Live Mesh Control Room

Recommended baseline. A three-pane cockpit balances live agent presence, trace chronology, and selected event detail. It prioritizes calm confidence and keeps actions close to context.
